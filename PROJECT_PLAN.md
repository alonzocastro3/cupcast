CupCast is a World Cup prediction and live intelligence dashboard.

Goal:
Build a production-style full-stack app in one weekend.

Stack:
- Frontend: Next.js, TypeScript, Tailwind, shadcn/ui
- Backend: FastAPI, Python
- Database: PostgreSQL
- Cache: Redis
- Deployment: Vercel + Render/Railway + Supabase/Neon

MVP:
- Dashboard with teams and matches
- Match prediction engine
- Team pages
- News feed
- Basic sentiment/trends
- Auth optional
- README, Docker, tests
